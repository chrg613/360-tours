# Roadmap

This folder describes the planned delivery phases for the 360 Tours Platform.

## Documents

- `mvp.md` — MVP scope, launch criteria, and acceptance gates.
- `ai-roadmap.md` — AI feature rollout plan (jobs, evaluation, governance).

## Roadmap rules

- The MVP is the smallest set of features that delivers the core value proposition with production-grade security and reliability.
- Post‑MVP items must not leak into MVP acceptance criteria.
